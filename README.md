# Desafio ­ Desenvolvedor PHP

Sistema para transportadoras

Os arquivos das tabelas do banco de dados estão na pasta:
/data/dump/

O arquivo de configuração está em:
/config/autoload/global.php

Instruções simples de instalação:

1. Criar as tabelas do banco de dados (Mysql);

2. Clonar o projeto numa máquina rodando Apache;

3. Ajustar as informações de conexão do banco de dados. Arquivo global.php

4. Apontar no no navegador para: http://localhost/admin

5. Login: administrador / Senha: 12345678
